Design And Simulation Of a Renewable Energy Based Power Conversion System

This project focuses on designing and simulating a renewable energy-based power conversion system for off-grid applications using solar energy. The system was developed to provide stable and reliable power to both DC and AC loads in areas where grid electricity is not available. The system includes a solar PV array, a DC–DC boost converter for voltage regulation, a full-bridge inverter for DC to AC conversion, an LC filter for waveform smoothing, and an AC voltage controller for RMS voltage adjustment. The entire system was modeled and simulated using MATLAB/Simulink to analyze voltage stability, waveform quality, and system performance under different conditions such as varying sunlight and load changes. This project demonstrates how renewable energy systems can be used to provide efficient and reliable electricity in remote areas.

Autonomous Fire Detection And Suppression With Environmental Monitoring System

Designed and developed an autonomous fire detection and suppression robot integrated with real-time environmental monitoring to enhance safety and enable early hazard response. The system uses an ESP32 microcontroller with multiple flame sensors to detect fire, automatically navigate toward the source, and extinguish it using a servo-controlled water pump mechanism. In parallel, an Arduino UNO monitors environmental conditions such as temperature, humidity, and combustible gases using DHT11, MQ2, and MQ6 sensors. The collected data is transmitted to a cloud platform (ThingSpeak) via Wi-Fi, allowing real-time monitoring and remote access through an online dashboard. The project demonstrates the integration of embedded systems, IoT connectivity, autonomous navigation, and sensor-based environmental monitoring, providing a low-cost and efficient fire safety solution suitable for smart homes, warehouses, and industrial environments.

Density Based Traffic Control System

A Density-Based Traffic Control System is an intelligent traffic management system that adjusts traffic signal timings based on the number of vehicles present on each road. Sensors are used to detect the traffic density at different lanes of an intersection. The system then gives priority to the lane with the highest vehicle density by allowing a longer green signal time. This helps reduce traffic congestion, minimize waiting time, and improve the overall flow of vehicles at busy intersections.

Arduino Uno Design Using ATmega328P

I designed and built a custom Arduino Uno-compatible board using the ATmega microcontroller and external components. The goal of this project was to understand the internal hardware design of Arduino and learn how microcontroller systems work at the circuit level. The circuit includes an ATmega microcontroller, a 16 MHz crystal oscillator for clock generation, 22 pF capacitors for oscillator stabilization, and a 10 kΩ resistor for reset functionality. The PCB layout was designed and prepared for fabrication. This project helped me understand microcontroller hardware design, clock circuits, reset circuits, and PCB layout design.

ESP32-Based Obstacle Avoidance Robot Using Ultrasonic Sensors

This project is an autonomous obstacle avoidance robot built using an ESP32 microcontroller and three ultrasonic sensors. The sensors continuously measure the distance between the robot and nearby objects. When an obstacle is detected, the system automatically decides the safest direction and changes the robot’s movement to avoid a collision. The use of three sensors helps the robot detect obstacles from multiple directions, improving navigation and safety. This project demonstrates the application of embedded systems, sensor integration, and autonomous navigation in robotics.
